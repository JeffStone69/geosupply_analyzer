Real-time ASX Shipping & Mining Rebound Analyzer with Grok (xAI) integration.

- Live rebound scores + RSI
- Shipping + Mining tickers
- Investment simulator
- Real xAI Grok API
- Self-updating

## Installation

streamlit run geosupply_rebound_analyzer.py
